# code-everyday
code-everyday is my personal daily development log where I commit code, document progress, practice new technologies, and work on real-world projects. The goal is to stay consistent, improve problem-solving skills, and grow professionally in software development.

## Monthly folders

Entries are organized by month under year-based folders:

- `2026/02-February/`
- `2026/03-March/`
- `2026/04-April/`
- `2026/05-May/`
- `2026/06-June/`
- `2026/07-July/`
- `2026/08-August/`
- `2026/09-September/`
- `2026/10-October/`
- `2026/11-November/`
- `2026/12-December/`

Each dated file stays unchanged inside its month folder.
